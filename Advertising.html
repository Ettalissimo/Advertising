I- L'algorithme de regression lineaire simple¶

L'algorithme de rgression linaire qu'on implmentera est bas sur la mthode des moindres carrs ordinaires (OLS), qui estime les coefficients du modle de rgression linaire en minimisant la somme des carrs des diffrences entre les valeurs observes et prdites.

1- Reprsentation du modle :

Le modle de rgression linaire simple peut tre reprsent comme suit :

$y = ax + b + $

o :

$y$ est la variable dpendante (cible),

$x$ est la variable indpendante (caractristique),

$b$ est l'interception (biais),

$a$ est la pente de la ligne,

$$ est le terme d'erreur.


2- Fonction de cot :

La fonction de cot minimiser est la somme des carrs des diffrences entre les valeurs observes et prdites :

$J(b, a) = \frac{1}{2m} \sum_{i=1}^{m} (h(x^{(i)}) - y^{(i)})^2$

$m$ est le nombre d'exemples d'entranement,

$h(x)$ est la valeur prdite pour l'entre $x$ donne par $ax+b$ .


3- quation normale :

L'quation normale fournit une solution en forme ferme pour les coefficients $a$ et $b$

$ (b,a) = (X^TX)^{-1}X^Ty$

o :

$X$ est la matrice des caractristiques (avec une colonne ajoute de 1 pour le terme d'interception),

$y$ est le vecteur des valeurs cibles.


Une fois que les coefficients $b$ et $a$ ont t estims, les prdictions peuvent tre faites en utilisant l'quation de rgression linaire.

II- Implementation de l'algorithme de regression lineaire simple¶

Ce code Python dfinit une classe LinearRegression qui effectue une rgression linaire simple en utilisant la mthode des moindres carrs. Il comporte des mthodes pour ajuster le modle aux donnes et pour effectuer des prdictions.

La mthode fit(self, X, y) ajuste le modle de rgression linaire aux donnes d'entranement fournies, tandis que la mthode predict(self, X) utilise ce modle pour effectuer des prdictions sur de nouvelles donnes.

import numpy as np

class LinearRegression:
    def _init_(self):
        self.a = None
        self.b = None

    def fit(self, X, y):
        # Ajouter une colonne de 1  X pour le terme constant
        X = np.hstack((np.ones((X.shape[0], 1)), X))

        # Calculer les coefficients via la formule des moindres carrs
        b, a = np.linalg.inv(X.T @ X) @ X.T @ y
        self.a = a  # La pente 'a' est le coefficient de X
        self.b = b  # L'interception 'b' est le coefficient constant

    def predict(self, X):
        # Calculer les prdictions
        predictions = self.a * X + self.b
        return predictions

Coefficient de dtermination (R): 0.2741325741814985

Un coefficient de dtermination (R) de 0.274 indique que le modle de rgression linaire explique environ 27.4% de la variation observe dans la variable cible (les ventes) partir de la variable explicative (le budget allou la publicit la radio).

Interprtation :

  • 27.4% est une valeur relativement faible pour le R. Cela signifie que le modle de rgression linaire avec uniquement le budget allou la publicit la radio comme caractristique explique seulement environ 27.4% de la variabilit des ventes. En d'autres termes, il existe d'autres facteurs non pris en compte par le modle qui influent sur les ventes.

  • Cela peut indiquer que le modle de rgression linaire utilis est trop simpliste pour capturer la relation entre le budget publicitaire et les ventes, et qu'il peut tre ncessaire d'inclure d'autres caractristiques ou d'utiliser des modles plus complexes pour amliorer la prdiction des ventes.

  • Par consquent, bien que le modle puisse fournir une certaine explication de la variation des ventes en fonction du budget publicitaire la radio, il est important de se rappeler qu'il y a encore une grande part de la variation des ventes qui n'est pas explique par le modle.
